# Group Theory — Interactive Visualisations

A collection of interactive web-based visualisations for exploring group theory, built for the [MATH0043 Mathematics for Physics and Astronomy](https://www.ucl.ac.uk/module-catalogue/modules/MATH0043) module at [UCL](https://www.ucl.ac.uk/) with lots of help from Claude and Gemini.

**🌐 Live site: [jt496.github.io/groups](https://jt496.github.io/groups/)**

## Overview

These visualisations let you explore examples and concepts from group theory through animated 3D models, Cayley graphs, and other interactive tools. Everything runs in the browser — no installation required.

## Structure

### ⬡ Plane Symmetries (`groups/plane/`)
- **Cyclic groups** — rotational symmetries Cₙ
- **Dihedral groups** — symmetries of regular polygons Dₙ
- **Rectangle symmetries** — the Klein four-group V₄
- **Frieze groups** — the 7 symmetry groups of frieze patterns
- **Wallpaper groups** — the 17 symmetry groups of the plane
- **Triangle groups** — groups generated by reflections

### ▲ Symmetries of Polyhedra (`groups/polyhedra/`)
- **Tetrahedron** — rotational (A₄) and full symmetry group (S₄)
- **Cube** — rotational (S₄) and full symmetry group (S₄ × C₂)
- **Dodecahedron** — rotational (A₅) and full symmetry group (A₅ × C₂)

### ⤨ Permutations & Integers mod n (`groups/algebra/`)
- **Permutations in Sₙ** — compose permutations, explore cycle notation
- **Addition mod n** — the additive group ℤ/nℤ
- **Units mod n** — the multiplicative group (ℤ/nℤ)×

### ⧉ Lagrange & Subgroups (`groups/structure/`)
- **Cosets & Lagrange's Theorem** — visualise coset partitions
- **Subgroup lattices** — Hasse diagrams of subgroup inclusion

### 🛠 Tools (`groups/structure/`)
- **Cayley graphs** — interactive graph explorer for any finite group
- **Group presentations** — generators and relations
- **Homomorphisms** — visualise group homomorphisms
- **Representations** — matrix representations of groups

### ✦ Other Topics (`groups/other/`)
- **Rubik's Cube** — the Rubik's Cube group

## Technology

Built with HTML, CSS, and JavaScript. 3D visualisations use [Three.js](https://threejs.org/). No build step — the site is a collection of static pages.

## Running Locally

Serve the `groups/` directory with any static file server:

```bash
cd groups
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).
